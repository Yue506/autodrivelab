# Legacy Prototypes

This folder keeps the original non-ROS prototypes for reference while the main implementation moves to ROS2 packages under `src/`.

- `adas_prototype/`: original ADAS arbitration, TTC and nuScenes-oriented prototype.
- `dms_prototype/`: original cabin DMS prototype.

Large runtime files, datasets, model weights, logs and `.docx` drafts are intentionally ignored for the public repository.
